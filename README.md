## 프로젝트 소개

널널 AI는 서울을 이동하는 사용자에게 목적지의 예상 혼잡도, 지하철 상황, 날씨, 주변 대체 장소를 한 화면에서 안내하는 서비스입니다.  
단순히 숫자를 보여주는 것이 아니라 “지금 가도 괜찮은지”, “언제 가면 덜 붐비는지”, “어디로 바꾸면 좋은지”를 바로 판단할 수 있게 돕는 것을 목표로 합니다.

현재 버전은 API 없이도 실제 서비스처럼 시연할 수 있도록 풍부한 mock 데이터를 기반으로 동작합니다.

## 핵심 기능

- 로그인/회원가입 및 소셜 로그인 형태의 진입 화면
- 홈 화면에서 목적지 검색 후 추천 경로와 혼잡 경고 확인
- 서울 지하철역 400개 이상의 mock 데이터 기반 지도 검색
- 혼잡도 순위 10위 및 역별 상세 정보 표시
- 일정 최적화 화면과 저장된 일정 관리
- 널널 AI 예측 그래프와 관광객 행동 가이드
- 오른쪽 아래 플로팅 AI 챗봇 비서
- GitHub Pages 배포를 고려한 정적 웹 export 구조

## 화면 구성

| 화면 | 설명 |
| --- | --- |
| 로그인 | 앱 첫 실행 시 가장 먼저 보이는 서비스 진입 화면 |
| 홈 | 목적지 입력, 추천 경로, 실시간 서울 현황 확인 |
| 지도 | 서울 지하철역 검색, 혼잡도 순위, 사실형 mock 지도 |
| 일정 | 여행 장소 추가 및 AI 순서 최적화 |
| 널널 AI | 홍대입구 혼잡도 예측 그래프와 행동 가이드 |
| 마이페이지 | 사용자 정보, 저장된 일정, 최근 검색 내역 관리 |

## 기술 스택

- Expo Router

- 정적 웹 export
- 프론트엔드 mock 데이터 기반 프로토타입

## 외부 참고 자료

OpenAI Skills 저장소의 파일을 프로젝트 내부에 직접 포함했습니다.

- 위치: `external/openai-skills`
- 원본: https://github.com/openai/skills.git
- 참고: submodule이 아니라 실제 파일로 포함되어 있어 별도 초기화 명령 없이 clone 후 바로 확인할 수 있습니다.

## 실행 방법

```bash
npm install
npm run web
```

정적 웹 빌드:

```bash
npm run export:web
node scripts/serve-static.js --port 8082
```

## 프로젝트 방향

널널 AI는 향후 서울시 실시간 인구 데이터, 지하철 혼잡도, 날씨, 길찾기 API와 연결해 더 정밀한 추천을 제공할 수 있습니다.  
현재는 발표와 시연을 위해 mock 데이터만으로도 전체 사용자 흐름이 자연스럽게 동작하도록 설계했습니다.
